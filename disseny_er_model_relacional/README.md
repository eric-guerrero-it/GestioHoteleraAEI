# Disseny ER i Model Relacional – Espamus+

Aquesta carpeta conté tota la documentació i fitxers relacionats amb el disseny de la base de dades del projecte **Gestió Hotelera Espamus+**, dins del projecte intermodular ASIX M0372-M0370-MS003.

## Contingut

- `model_er_hotel.pdf': Diagrama Entitat-Relació (ER) creat amb dbdiagram.io segons els requisits del projecte.
- `model_relacional.txt`: Model relacional complet amb comentaris i justificació de claus primàries, foranes, i relacions N:M o especialitzacions.
- `script_creacio_taules_bd_espamus.sql`: Script SQL de creació de totes les taules de la base de dades amb restriccions i referències.
- `Disseny_ER_Model_Relacional_Espamus_COMPLET.pdf`: Document explicatiu amb objectius, descripció i estructura del model.

## Objectiu

L’objectiu d’aquesta fase és definir amb precisió la base de dades del sistema de gestió hotelera. El model inclou:

- Tractament de clients, treballadors i especialitzacions (cuina, recepció…)
- Assignació d’habitacions i reserves
- Gestió de serveis oferts per l’hotel
- Sol·licituds de serveis i facturació
- Està preparat per suportar multilingüisme i estructura escalable
